Power BI Analytics Dashboards: 
    This repository contains end-to-end **Power BI dashboards** created to analyze business performance, customer behavior, and operational metrics using real-world datasets.  
The dashboards are designed with an **executive and business-focused approach**, emphasizing clear KPIs and actionable insights.

Project 1: Swiggy Food Delivery Analytics Dashboard
    An executive-level Power BI dashboard built using a **50K+ row food delivery dataset** to analyze revenue, orders, customers, and restaurant performance for a food delivery platform.

 Key KPIs
- Total Revenue  
- Total Orders  
- Unique Customers  
- Average Rating  
- Monthly Active Customers  

 Key Analysis & Insights
- Analyzed **monthly revenue trends** to identify growth patterns and seasonality  
- Evaluated **order status performance** (Delivered vs Cancelled)  
- Assessed **customer behavior** through payment method distribution  
- Performed **city-wise revenue analysis** to identify high-performing regions  
- Identified **top-performing restaurant partners** driving platform revenue



Project 2: Uber Executive Performance Dashboard
Overview:
A Power BI dashboard designed to analyze **trip performance, revenue trends, and driver metrics** for a ride-hailing platform.

Key KPIs
- Total Trips  
- Total Revenue  
- Average Fare  
- Active Drivers  

Key Analysis & Insights
- Analyzed **trip and revenue trends** over time  
- Evaluated **city-wise performance**  
- Assessed **payment method distribution**  
- Ranked **top drivers by revenue contribution**  
- Designed insights for **executive-level decision-making**
